# Python Fundamentals

This repository contains **48 Python practice files** covering all fundamental topics from A to Z.  
It is designed for anyone who wants to strengthen their Python skills through hands-on practice.

---

### 📂 Repository Structure
- **Basics:** Variables, Data Types, Input/Output, Type Casting.
- **Operators:** Arithmetic, Assignment, Comparison, Logical.
- **Control Structures:** If-elif-else, For/While Loops, Break/Continue.
- **Data Structures:** Lists, Arrays, Tuples, Dictionaries, Sets.
- **Functions:** User-defined functions, Lambda, Recursion.
- **Object-Oriented Programming (OOP):** Classes, Objects, Inheritance, Abstraction.
- **Advanced:** Datetime, Exception Handling, File I/O.

### 🛠 Prerequisites
Before running the code, make sure you have:
* Python 3.10+ installed.
* An IDE like PyCharm or VS Code (optional but recommended).
